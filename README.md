# Tutorial React Native
Tutorial membuat aplikasi mobile dengan menggunakan React Native Expo
***

## Included
* React & React Native
* React Navigation
* Expo

## Instalasi
See [Getting Started](https://reactnative.dev/docs/getting-started.html) to install requirement tools.

 ```$ npm install```

## Development
### Start local server

```$ npm start```

### iOS

```$ npm run ios```

### Android

```$ npm run android```
